# Documentation Assets

This folder contains visual assets and guides for the project.

## Files

- **demo.gif** - Animated demo showing real-time collaboration (add this file)
- **RECORDING_GUIDE.md** - Instructions for recording the demo

## Adding the Demo GIF

1. Record your screen following the guide in `RECORDING_GUIDE.md`
2. Save the GIF as `demo.gif` in this folder
3. Optimize it to keep under 10MB: https://ezgif.com/optimize
4. Commit and push - it will appear in the README automatically!
